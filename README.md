# Tureng-rs

Local [Tureng](https://tureng.com) client. Tureng is a translation service that can translate between these languages:
* Turkish - English
* German - English
* Spanish - English
* French - English

## Usage
```console
$ tureng-rs --interactive --lang enfr
$ tureng-rs schmeckt --lang ende
```
