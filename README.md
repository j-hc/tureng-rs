# Tureng-rs

A [Tureng](https://tureng.com) client in terminal. Tureng is a translation service that can translate words for these languages:
* Turkish - English
* German - English
* Spanish - English
* French - English

## Usage
```console
$ tureng-rs --interactive --lang enfr
$ tureng-rs word
```
